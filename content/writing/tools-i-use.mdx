People often ask what tools I use. Here's a living list of software and hardware I rely on daily.

## Editor & Development

- **VS Code** with a minimal theme — I keep extensions to a minimum
- **iTerm2** + **zsh** — fast terminal setup
- **Git** — version control for everything, even writing

```bash
# My git alias for quick commits
alias gc="git add -A && git commit -m"
```

## Writing

- **MDX files** in my code editor — I write posts the same way I write code
- **Grammarly** for a quick proofread
- **Bear** for draft notes on mobile

## Design

- **Figma** — for anything visual
- **Excalidraw** — for quick diagrams and whiteboarding
- **Linear** — for project management

## Hardware

- MacBook Pro (M-series) — everything I need in one machine
- Studio Display — clean, sharp, and plenty of real estate
- Mechanical keyboard — the tactile feedback matters more than you'd think

<Callout type="note">
I'll update this list as my setup evolves. Tools change, but principles stay the same: choose things that get out of your way.
</Callout>

## Philosophy

The best tool is the one you actually use. I optimize for speed, simplicity, and reliability over feature count.
